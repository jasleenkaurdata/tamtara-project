# Tamtara Restaurant Website

## Overview

Tamtara is a modern restaurant website designed to showcase menu offerings, restaurant information, and customer contact details through a clean and responsive user experience.

This project was initially generated using AI-assisted development tools and was further customized through manual editing, configuration, and frontend development work.

## Features

- Responsive design for desktop and mobile devices
- Restaurant menu showcase
- Contact and location information
- Modern user interface
- Fast-loading frontend experience

## Technologies Used

- JavaScript

- Visual Studio Code
